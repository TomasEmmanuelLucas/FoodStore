# Food Store

## Descripción

Food Store es una aplicación desarrollada en Java utilizando Programación Orientada a Objetos (POO). El sistema permite administrar categorías, productos, usuarios y pedidos mediante un menú interactivo por consola.

El proyecto fue desarrollado aplicando los conceptos vistos en la materia, tales como herencia, encapsulamiento, interfaces, excepciones personalizadas, enumerados y colecciones dinámicas.

Requisitos
Java JDK 17 o superior.
Apache NetBeans IDE.

Ejecución
Abrir el proyecto en Apache NetBeans.
Esperar a que el IDE cargue todas las dependencias.
Ejecutar la clase Main ubicada en:

src/FoodStore/ui/Main.java

Utilizar el menú por consola para acceder a las distintas funcionalidades del sistema:
Categorías
Productos
Usuarios
Pedidos
